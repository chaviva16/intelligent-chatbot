## Intelligent Chat Assistant

A friendly AI chatbot powered by **Google Gemini**.  
This chatbot can answer questions, tell jokes, provide dates, explain coding concepts, and more.

## Features
- Conversational AI chatbot
- Gemini-powered responses
- Friendly and helpful
- Written in Python using Streamlit



# Usage

Type your message in the input box and press Send.

Chat with the AI about anything: coding, jokes, dates, explanations, etc.
Use Clear Chat to reset the conversation.

# Notes
Keep your .env file secret. Do not upload it to GitHub.

This project uses Gemini API; ensure your API key is valid.
